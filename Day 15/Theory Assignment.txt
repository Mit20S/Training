3. Difference between ==  and ===?
----    == compares the value of two variable
        === compares the data type as well as the value of the variable.

4. What does the isNaN() function?
----    isNaN() function checks if the variable is a number.
        If yes, the value of variable is a number, then isNaN() = false
        If no, the value of variable is not a number, then isNaN() = true
        
5. Difference between Client side JavaScript and Server side JavaScript?
----    Client-side javascript allows you to make possible some user interactions, by for instance changing the    DOM, adding events to html elements, etc..
Server side javascript, allows to exploit the javascript in its entireness by building more scalable, event-driven and non I/O blocking applications.
